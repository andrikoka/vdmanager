Szótar

  • cross-platform - népszerűbb operációs rendszereken használható (windows, linux, mac)
  • fájlasszociáció - egy adott fájltípus "társítása" egy azt megjeleníteni, használni képes programmal, programcsomaggal
  • API* - Application Programming Interface - egy közös programozási felület, mely funkcióit és szolgáltatásait ugyanazon felületen keresztül érhetjük el, a konkrét megvalósítás ismerete nem szükséges
